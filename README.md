# Cardiovascular Disease Prediction

A Machine Learning project to predict the presence of cardiovascular disease 
using patient health data.

## 📌 Project Overview
Heart disease is one of the leading causes of death worldwide. This project 
applies multiple ML algorithms to predict cardiovascular disease based on 
patient attributes like age, blood pressure, cholesterol, BMI, and lifestyle factors.

## 📊 Dataset
- Source: cardio_train.csv
- Features: Age, Gender, Height, Weight, Blood Pressure, Cholesterol, Glucose, Smoking, Alcohol, Activity

## ⚙️ Models Used
- Support Vector Machine (SVM)
- K-Nearest Neighbor (KNN)
- Decision Tree (DT)
- Logistic Regression (LR)
- Random Forest (RF)

## 📈 What's Included
- Data Pre-processing & Cleaning
- Exploratory Data Analysis (EDA)
- 11 Visualisation Plots
- Correlation Matrix
- Model Training & Accuracy Comparison
- Confusion Matrix & ROC-AUC Curve
- Feature Importance Analysis

## 🛠️ Technologies Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 🚀 How to Run
1. Clone the repository
2. Place `cardio_train.csv` in the project folder
3. Run: `python cardiovascular_disease_prediction.py`
